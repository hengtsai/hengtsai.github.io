用VScode寫Markdown，快速插入圖片

Posted on 2022-06-10 Views: Disqus:

自己架設靜態blog，全部都要自己搞。文章評論區、統計瀏覽次數，沒有即時易用的線上文章編輯器只能用markdwon、sitemap、連圖床也要自己找。好不容易找到了圖床之後，還要先上傳圖片，才能把圖片連結網址貼到markdwon上，才有圖文並茂的文章。光想想就很不人性化。這邊做一下筆記，記錄自己上網找的VScode plugin以及設定方法。以下是vscode-imgur的demo，看起來就很吸引人。

VScode + Markdown Image 插圖超方便

但我自己是選用Markdown Image的好處有幾個：

  1. 支持各種空間，例如local (github)、Imgur、Cloudinary、Cloudflare…等。
  2. 支援自動更改檔名，例如日期-文章名稱-編號。不支持 Imgur 和 SM.MS。
  3. 自動轉換成Markdown語言貼上。

Cloudinary 設定

我自己是用Cloudniary設定。因為他可以上傳比較多種類的檔案。並且一個月也有25GB的流量，應該是很夠我使用。

申請完帳號後，在Dashboard就可以看到必要資訊，依序填入Markdown Image的設定裡。其中，資料夾我試了一下不能空白，一定要填個資料夾。如果server上沒有他會自己創建。

Imgur 設定

Imgur要先去申請去申請一個API application。

按照以上填入申請單之後，就會出現的Client ID and Secret。如果API key資訊不見的話，也可以到Imgur點選右上角的帳號-settings-Applications就可以看到你的Client Name / Client ID / Client Secret。

不過如果到VScode Markdown Image的設定看，會發現他只需要輸入Client ID，不用填Client Secret。這代表你所上傳的圖片並不會真的出現在你帳號裡。這是Markdown Image不好用的地方。如果專攻Imgur的話，還可以參考另一個軟體vscode-imgur。
